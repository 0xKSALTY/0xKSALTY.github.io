About Me

Operating Systems

Simple Backup Script

This script uses basic commands to create a compressed archive of the 'work' directory. It echoes a message to indicate the start and end of the backup process.

Timestamped Backup Script

This version of the script enhances the previous one by adding a timestamp to the backup file name. It ensures each backup is unique and can be easily identified by the date it was created.

Parameterized Backup Script

This script introduces a parameter for the source directory, making it flexible to back up any directory specified by the user when executing the script.

Multiple Directories Backup Script

This advanced script iterates over a list of directories, creating separate backup files for each. It combines the features of timestamping and parameterization for robust functionality.
